# 🍜 ครัว 3 ป. - เว็บไซต์ร้านอาหาร

เว็บไซต์ร้านอาหารตามสั่ง "ครัว 3 ป." ที่แอดมินสามารถอัพเดทเนื้อหาได้ทุกจุดผ่านแพนเนลจัดการบนเว็บไซต์ รองรับการใช้งานบนมือถืออย่างเต็มรูปแบบ

## 📁 โครงสร้างโปรเจกต์

```
krua-3-por-website/
├── index.html          # หน้าหลัก
├── css/
│   └── style.css       # สไตล์ชีต
├── js/
│   ├── config.js       # ไฟล์ตั้งค่า (แก้ไขได้)
│   └── app.js          # แอปพลิเคชันหลัก
└── images/             # โฟลเดอร์รูปภาพ (ว่าง)
```

## 🚀 วิธีเปิดใช้งานใน VS Code

### วิธีที่ 1: Live Server (แนะนำ)
1. ติดตั้ง Extension **"Live Server"** ใน VS Code
2. คลิกขวาที่ไฟล์ `index.html` → เลือก **"Open with Live Server"**
3. เว็บไซต์จะเปิดที่ `http://127.0.0.1:5500`

### วิธีที่ 2: เปิดไฟล์ตรง
1. ดับเบิลคลิกที่ไฟล์ `index.html`
2. เว็บไซต์จะเปิดในเบราว์เซอร์เริ่มต้น

## ⚙️ วิธีอัพเดทเนื้อหาเว็บไซต์

### วิธีที่ 1: แพนเนลจัดการบนเว็บไซต์ (แนะนำ - ใช้บนมือถือได้)
1. เปิดเว็บไซต์
2. คลิกปุ่ม **⚙️ (ฟันเฟือง)** ที่มุมซ้ายล่าง
3. เลือกแท็บที่ต้องการแก้ไข:
   - **ทั่วไป** - ชื่อร้าน, คำขวัญ, ลิขสิทธิ์
   - **หน้าแรก** - หัวข้อ, คำบรรยาย, เบอร์โทร, ที่อยู่, รูปพื้นหลัง
   - **เมนู** - เพิ่ม/ลบ/แก้ไขรายการเมนูอาหาร
   - **เกี่ยวกับ** - เนื้อหา, รูปภาพ, สถิติ
   - **ติดต่อ** - เบอร์โทร, ที่อยู่, เวลาเปิด-ปิด, แผนที่
   - **ไฮไลท์** - จุดเด่นของร้าน
   - **แกลเลอรี่** - รูปภาพและคำอธิบาย
   - **โซเชียล** - ลิงก์ Facebook, Instagram, Line, TikTok
4. แก้ไขข้อมูลตามต้องการ
5. คลิก **"บันทึกการเปลี่ยนแปลง"**
6. ข้อมูลจะถูกบันทึกในเบราว์เซอร์ (localStorage)

### วิธีที่ 2: แก้ไขไฟล์ config.js โดยตรง
1. เปิดไฟล์ `js/config.js` ใน VS Code
2. แก้ไขค่าต่างๆ ในออบเจกต์ `SITE_CONFIG`
3. บันทึกไฟล์ → รีเฟรชหน้าเว็บ

### วิธีที่ 3: แก้ไข HTML โดยตรง
- เปิดไฟล์ `index.html` ใน VS Code
- แก้ไขเนื้อหาโดยตรงใน HTML

## 📱 ฟีเจอร์เด่น

- ✅ **Responsive Design** - แสดงผลสวยงามทั้งบนมือถือ, แท็บเล็ต, และคอมพิวเตอร์
- ✅ **Admin Panel** - แก้ไขเนื้อหาทุกจุดผ่านหน้าเว็บ ใช้งานบนมือถือได้
- ✅ **เมนูอาหารแบบแท็บ** - กรองเมนูตามหมวดหมู่
- ✅ **ปุ่มโทรลอย** - ลูกค้ากดโทรสั่งอาหารได้ทันที
- ✅ **แผนที่ Google Maps** - แสดงตำแหน่งร้าน
- ✅ **แกลเลอรี่รูปภาพ** - แสดงบรรยากาศและเมนูอาหาร
- ✅ **Animation** - เอฟเฟกต์เลื่อนเข้ามาเมื่อเลื่อนหน้า
- ✅ **Loading Screen** - หน้าโหลดสวยงาม
- ✅ **Dark Navbar** - แถบนำทางเปลี่ยนสีเมื่อเลื่อน

## 🎨 ธีมสีที่ปรับแต่งได้

ใน `config.js` สามารถปรับเปลี่ยนสีได้:
```javascript
theme: {
    primaryColor: "#e74c3c",    // สีหลัก (แดง)
    secondaryColor: "#f39c12",  // สีรอง (ส้ม)
    accentColor: "#27ae60",     // สีเน้น (เขียว)
    darkColor: "#2c3e50",       // สีเข้ม
    lightColor: "#ecf0f1",      // สีอ่อน
    textColor: "#333333"        // สีตัวอักษร
}
```
